# *Công nghệ phần mềm:*

### *Software product:* Sản phẩm phần mềm
- **`Là một tập hợp, gồm 3 phần:`**
	- `Chương trình máy tính:`
		- `Source code (mã nguồn), executable code (mã thực thi).`
		- `Machine code, mã máy.`
	- `Cấu trúc dữ liệu:`
		- `Memory: Bộ nhớ trong.`
		- `Storage: Bộ nhớ ngoài.`
	- `Tài liệu liên quan (có nhiều loại nhưng chỉ quan tâm tài liệu kĩ thuật):`
		- `SRS - Software Requirements Specification: Đặc tả yêu cầu.`
		- `SDD - Software Design Document: Thiết kế.`
		- `API Documentation.`
		- `Test Plan/Test Cases.`
		- `BRD - Business Requirements Document: Yêu cầu nghiệp vụ.`
- **`Phân loại phần mềm sản phẩm:`**
	- `Sản phẩm dùng chung.`
	- `Sản phẩm đặt hàng.`
	- `Phần mềm hệ thống.`
	- `Phần mềm nghiệp vụ.`
	- `Phần mềm công cụ.`
- **`Mô hình chất lượng:`**
	- `Tính năng: Phù hợp với người dùng.`
	- `Tính tin cậy: High availability, fault tolerance, resilience.`
	- `Tính khả dụng: Dễ dùng, dễ hiểu, áp dụng nhiều việc đối với người dùng.`
	- `Tính hiệu quả: Performance and resource preservation.`
	- `Khả năng bảo trì: Readable, OOP, ...`
	- `Tính khả chuyển: Portable.`

### *Information system:* Hệ thống thông tin
- **`Là tập hợp các thành phần phối hợp thu thập thông tin hỗ trợ việc ra quyết định:`**
	- `Phần cứng.`
	- `Phần mềm.`
	- `Dữ liệu.`
	- `Thủ tục.`
	- `Con người.`
- `Công cụ = phần cứng + phần mềm.`
- `Nguồn lực = thủ tục + con người.`
- `Cầu nối = Dữ liệu.`
