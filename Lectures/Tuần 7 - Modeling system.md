### Mô hình hóa hệ thống:

- `Góc nhìn bên ngoài:`
	- `Ngữ cảnh, môi trường hệ thống.`
- `Góc nhìn hoạt động:`
	- `Luồng hoạt động của hệ thống.`
- `Góc nhìn tương tác:`
	- `Giữa môi trường và hệ thống, giữa các bộ phận hệ thống.`
- `Góc nhìn cấu trúc:`
	- `Tổ chức hệ thống, cấu trúc dữ liệu.`
- `Góc nhìn hành vi:`
	- `Hành vi động của hệ thống, hành vi phản hồi của hệ thống.`

### Những model tư duy - UML diagram types:

- `UML diagram:`
	- `5 kiểu UML diagram:`
		- `Use Case Diagram.`
		- `Activity Diagram.`
		- `Sequence Diagram.`
		- `Class Diagram.`
		- `State Machine Diagram.`
	- `Mỗi kiểu sẽ dùng để thể hiện tư duy mô hình tương ứng bên dưới.`

- `Context model:`
	- `Mô hình ngữ cảnh cho biết ngữ cảnh xung quanh hệ thống.`
	- `Implementation: Use case diagram.`

- `Process model:`
	- `Mô hình quy trình.`
	- `Implementation: Activity diagram.`

- `Interaction model:`
	- `Mô hình tương tác.`
	- `Implementation: Sequence diagram, use case diagram.`

- `Structural model:`
	- `Mô hình cấu trúc.`
	- `Implementation: Class diagram.`

- `Behavioral Model:`
	- `Mô hình hành vi.`
	- `Implementation: State Diagram`
