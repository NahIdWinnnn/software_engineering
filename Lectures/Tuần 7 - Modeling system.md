# *Các góc nhìn hệ thống:*

- `Góc nhìn bên ngoài: Ngữ cảnh, môi trường hệ thống.`
- `Góc nhìn tương tác: Giữa môi trường và hệ thống, giữa các bộ phận hệ thống.`
- `Góc nhìn cấu trúc: Tổ chức hệ thống, cấu trúc dữ liệu.`
- `Góc nhìn hành vi: Hành vi động của hệ thống, hành vi phản hồi của hệ thống.`

# *Mô hình ngữ cảnh:*
- `Xác định biên hệ thống:`
	- `Những gì thuộc về hệ thống.`
	- `Những gì thuộc về môi trường.`
- `Mô hình quy trình:`
	- `Miêu tả tương tác hệ thống với môi trường trong ngữ cảnh.`
	- `Có thể dùng UML để mô hình hóa quy trình nghiệp vụ.`
